Для корректной работы бота необходимо сначала запустить main.py, а затем уже добавлять его в группы (уже созданные, это важно, если создать группу сразу с ботом, он её не увидит)
Бот знает и видит только тех юзеров, которые вступили (были приглашены) в группы, где уже работает бот (хоть в тихом режиме), в противном случае они будут недоступны для бана

Нужно узнать id человека (idbot), который будет писать команды боту в ЛС, и добавить его в config.py
Если работа бота была остановлена вручную (на сервере), за это время не добавляйте/кикайте его из групп, он загрузится из config.json, где там такого не будет
